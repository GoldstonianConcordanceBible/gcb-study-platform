# Content Generation Note

The GCB Study Platform uses generator-driven content scaffolding so the system can scale across canon, curriculum, scripture, podcasts, articles, and multilingual layers without manual file-by-file rebuilding.

## Purpose

This approach preserves:

- stable IDs
- stable slugs
- predictable app routing
- search compatibility
- curriculum alignment
- future multilingual support
- public build transparency

## Layers covered

- canon
- playlists
- courses
- reading plans
- reflection questions
- articles
- podcasts
- languages
- bible reader text scaffold

## Editorial rule

Generated content is structural content first.

It can be enriched later with:

- deeper summaries
- linked playlists
- chapter expansion
- doctrine cross-links
- Substack URLs
- transcript URLs
- multilingual text
