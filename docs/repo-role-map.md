# Repo Role Map

## Repository
`gcb-study-platform`

## Role
This repository serves as the curriculum and app layer of the Goldstonian Concordance Bible ecosystem.

## Primary Functions

### 1. Canon Navigation Layer
Organizes the 81-Book Ethiopian Canon into structured entries for study, retrieval, and application interfaces.

### 2. Playlist Mapping Layer
Maps YouTube playlists and videos to books, themes, and doctrines.

### 3. Doctrine Layer
Defines the Mirror → Water → Fire Doctrine and supporting theological categories.

### 4. Curriculum Layer
Converts media archives into study plans, reading plans, and courses.

### 5. App Content Layer
Acts as the source content for a future website and study application.

### 6. Donor Visibility Layer
Shows ongoing public progress for supporters, collaborators, and future institutional reviewers.

### 7. GEO Layer
Supports Generative Engine Optimization by organizing stable, high-quality theological entities and relationships.

## Relationship to Other Repositories

- `canonical-index` = primary machine-first canonical authority
- `gcb-study-platform` = application and curriculum layer
- `GCB-Knowledge-Infrastructure` = broader knowledge architecture and ontology
